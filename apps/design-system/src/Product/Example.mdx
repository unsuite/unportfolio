# Pagina decisione (esempio)

Una pagina decisione **viva**: il markdown qui sotto è il template di
`docs/product/decisions/`, importato `?raw` e renderizzato qui. Copia questo
file in `src/Product/<Decisione>.mdx`, punta l'import al tuo PDR e incolla il
prototipo. (La skill `product-decision` fa entrambe le cose.)
